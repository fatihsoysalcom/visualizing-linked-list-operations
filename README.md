# Visualizing Linked List Operations

This Python script demonstrates the fundamental operations of a singly linked list: appending, inserting, and deleting nodes. It provides a textual visualization of the list's state after each operation, illustrating how data structures change dynamically.

## Language

`python`

## How to Run

Run the script using a Python interpreter: `python linked_list_visualizer.py`
